4.x的java8分支，COLUMN模式和NONE模式才有Activiti工作流，DATASOURCE模式没有。

已激活的流程定义就可以在代码层面通过流程标识发起流程了。

![](/images/intro/基础平台_流程部署.png)

## 功能介绍

1. 上传

   上传模型压缩包

2. 转换模型

   就流程转换为可在线编辑的模型

3. 挂起

   修改流程的状态

4. 流程图

   在线查看流程图

5. XML

   在线查看流程的XML定义

6. 删除

   删除流程
