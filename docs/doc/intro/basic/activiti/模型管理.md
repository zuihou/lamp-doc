在线绘制工作流模型。



## 功能介绍

1. 新增

   新增一个空模型

2. 编辑

   在线制作流程图

   ![](/images/intro/基础平台_模型管理_编辑.png)

3. 发布

   将制作好的流程图发布为流程部署中。

4. 下载

   下载模型数据

5. 删除

   删除模型
