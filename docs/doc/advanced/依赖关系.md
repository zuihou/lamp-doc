## 独立编译顺序

由图可知： 依赖关系为： lamp-oauth -> lamp-generator -> lamp-base -> lamp-system
若想将lamp-cloud项目按模块独立编译，编译顺序为：

- lamp-dependencies-parent
- lamp-public
- lamp-system
- lamp-base
- lamp-generator
- lamp-oauth
- lamp-gateway
- lamp-support
